# AsyncValue
복잡한 요구사항들을 처리하기 위해서 상세하게 알아둘 필요가 있음

## important properties
- value
- error, stackTrace
- isLoading
- hasValue
- hasError (AsyncValueX)
- isReloading
- isRefreshing

## important methods
사용 예제는 AsyncValue 예제 참조

- `when()`
- `maybeWhen()`
- `whenOrNull()`
- `map()`
- `copyWithPrevious()`
